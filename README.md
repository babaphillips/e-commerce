## Installation

1. Clone the repo
   ```sh
   git clone git@github.com:babaphillips/e-commerce.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Install Inquirer package
   ```sh
   npm i inquirer@8.2.4
   ```
4. Install MySQL2
   ```sh
   npm install --save mysql2
   ```
5. Install Express

   ```sh
   npm install express

   ```

6. Initiate application
   ```sh
   node server
   ```
